About QC4LAB

The QC4LAB web app is a tool for checking agreement and plotting between two different values. Users can explore the data with flexible options. The multiple regression methods can be applied to estimate intercept and slope under different assumptions. The plots (Scatter plot & Bland-Altman plot) can be saved as a PNG file or a PDF file, which can be opened and edited with Adobe Illustrator to allow for fine adjustments of the lay-out.

3. Technical reference

3-1. Regression methods

All of these methods is a technique to fit a straight line in two-dimensional dataset.
  • Least square regression: This is simple linear regression where only the response variable (Y) is measured with error.
  • Deming regression: This regression method was introduced by Deming WE in 1943. It takes into account for errors on both the exposure (X) and the response variable (Y).
  • Passing-Bablok regression: This regression method was introduced by Passing H & Bablok W in 1983. It can be performed without any specific assumptions of both the exposure (X) and the response variable (Y) (distribution & measurement errors).

3-2. Bland-Altman plot

This plot is introduced by Bland JM and Altman DG in 1983. This plot is useful to explore the difference between two measurements.
For example, if you change an instrument for a specific biomarker in laboratory, you need to check the difference of values from instrument A with instrument B.
